# Lucide React

## Версия в проекте
`0.541.0`

## Описание
Lucide - это набор красивых и консистентных иконок для React.

## Usage

```typescript
import { Check, X, Trash2, Plus } from 'lucide-react';

<Check className="h-4 w-4" />
<X className="h-4 w-4 text-red-500" />
<Trash2 size={16} />
<Plus strokeWidth={2.5} />
```
